用户讨论以下话题时使用本 skill：

- 终末地 / Arknights Endfield 的剧情情节、世界观、角色关系
- 特定干员（如 Baker、Perlica、Chen Qianyu 等）的档案、语音、出场剧情

- 用户引用 Warfarin Wiki 内容或要求查证某段剧情来源

## 检索工作流

按以下顺序检索，**不要一上来就 grep 全库**——优先用索引和分类目录定位，再读原文。

### Step 1: 用条目索引定位

```bash
grep -i "关键词" 终末地剧情/_条目索引.md
```

索引行格式：`- [条目名](分类/文件路径.md) | slug | [来源](Wiki URL)`，命中后直接读出文件路径。

适用于：用户提到了明确的条目名（干员名、章节名、武器名、档案名）。

### Step 2: 世界观/概念性问题先读分析

```bash
ls 终末地剧情/分析/
```

`分析/` 下有 7 篇人工整理的结构化分析（时间线、势力关系、核心概念等），是了解世界观脉络的最佳入口。读完分析后再按其证据链反查原文。

适用于：用户问"X 是什么"、"X 和 Y 是什么关系"、"X 的来历"这类需要综合的问题。

### Step 3: 在具体分类目录里 grep 原文

```bash
grep -r "关键词" 终末地剧情/见闻辑录/
grep -rl "关键词" 终末地剧情/任务/   # 只列文件名
```

适用于：用户问的是具体台词、具体场景、具体物件，且不知道在哪个条目里。**务必指定分类目录**而非全库 grep，否则结果太多。

### Step 4: 读原文

定位到文件后用 Read 工具读完整内容。剧情类文件不大（通常 < 100KB），可以一次读完。

## 引用规范

在回答中引用剧情内容时：

- 注明条目名 + 分类，例如：「据《中枢档案·塔卫二》」「《见闻辑录·静默之主》」「任务 e1m1《基地解围》」

- 区分游戏剧情（已发生）和人工分析（解读）；引用 `分析/` 下内容时明确说明这是整理者的解读
