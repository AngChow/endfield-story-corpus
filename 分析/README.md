## 引用规则

本文档只使用本地文本库中的条目，不引入外部网页或库外设定。正文中的“依据”链接均指向本库原文，例如：[泰拉](<../中枢档案/0019_nar_document_v0d8_1_1_泰拉.md>)。

结论段统一采用证据先行格式：`依据：A 条目说明……；B 条目说明……；C 任务中某角色提到……。据此可判断……`。实际写作时，A/B/C 都应使用指向本库原文的相对链接。若是推论，收束语改为“据此可推断……”或“倾向于说明……”。若证据不足，则明确写入“未知”或“尚不能确定”。

为了便于判断可靠度，分析中使用三种标记：

- **事实**：原文直接给出的信息，通常由中枢档案、任务正文或见闻辑录直接支撑。
- **推论**：多个条目组合后得出的解释，会写明“可推断”“倾向于说明”等表述。
- **未知**：文本留下的问题或暂时无法证实的内容。
