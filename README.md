# 明日方舟：终末地 本地文本库

这个目录保存从 Warfarin Wiki 采集的《明日方舟：终末地》简体中文文本，按 `任务`、`Baker`、`干员`、`武器`、`中枢档案`、`见闻辑录` 六类组织。

## 使用

- 人工阅读：从 `_条目索引.md` 进入对应条目。
- 全文检索：搜索 `_说话文本索引.md` 或四个分类目录。
- 更新数据：运行 `python tools/fetch_warfarin_texts.py`。

## 目录

- `任务/`：任务 Transcript 与 Radio。
- `Baker/`：Baker 会话和全分支文本。
- `干员/`：干员情报、档案和语音记录。
- `武器/`：武器概览描述和武器描述文本。
- `中枢档案/`：中枢档案正文。
- `见闻辑录/`：见闻辑录正文。
- `_条目索引.md`：所有条目的入口。
- `_说话文本索引.md`：按行汇总的检索索引。
- `_来源与采集.md`：来源、采集结果和失败列表。
